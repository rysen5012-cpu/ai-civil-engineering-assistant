# AI Civil Engineering Assistant

An AI-powered assistant built using Google Gemini to help civil engineering students understand concepts easily.

---

## 🚀 Features

- Concept explanation (Dams, Concrete, etc.)
- Step-by-step procedure generation
- Automated report generation
- Structured and student-friendly answers

---

## ⚙️ How it Works

User Input → Gemini AI → Structured Output

The system takes a question and generates clear, organized responses using AI.
